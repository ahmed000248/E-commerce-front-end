# E-commerce-front-end

Frontend eCommerce internship project using HTML, CSS, and JavaScript (Figma-based design)
🛒 Ecommerce Frontend Design (Internship Project)
📌 Project Overview

This is a frontend eCommerce website project developed as part of a frontend development internship.
The project is based on a Figma UI design and focuses on building a pixel-perfect desktop version using HTML, CSS, and JavaScript.

It includes multiple pages such as Home, Product Listing, and Product Details with basic interactivity and clean UI structure.

🎯 Objective
Convert Figma design into a real working website
Practice HTML, CSS, and JavaScript fundamentals
Implement responsive desktop layout using Flexbox and Grid
Build reusable and clean UI components
🛠️ Technologies Used
HTML5 (Semantic Structure)
CSS3 (Flexbox, Grid, Styling)
JavaScript (Basic Interactivity)
Git & GitHub (Version Control)
Figma (UI Design Reference)
📄 Pages Included
🏠 Home Page
Hero banner section
Featured categories/products
Call-to-action buttons
🛍️ Product Listing Page
Grid-based product cards
Product image, name, price
“Buy Now” button
Hover effects and layout styling
📦 Product Details Page
Large product image
Product description & price
Add to Cart button
Related products section
Basic dropdown (size selection)
⚙️ Features
Pixel-perfect UI based on Figma design
Clean and structured code
Reusable CSS components
Hover effects for better UX
Basic JavaScript interactivity (dropdown, UI behavior)
